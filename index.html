Частые вопросы

На каких ОС работает программа?

Мы поддерживаем Windows 10/11, macOS (Intel и Apple Silicon M1/M2/M3) и Linux (Ubuntu, Debian через AppImage).

Нужны ли прокси для работы?

Да, для массовой работы мы настоятельно рекомендуем использовать качественные HTTP прокси. Это защитит ваши аккаунты от блокировок.

Есть ли бесплатная версия?

Да, вы можете скачать DEMO версию бесплатно. Она позволяет проверить весь основной функционал программы перед покупкой.
